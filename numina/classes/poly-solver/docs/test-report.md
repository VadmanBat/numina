# PolySolver — Отчёт по тестированию

**PolySolver** (Implicit Deflation + Laguerre + Newton polishing) был протестирован на 10 стандартных группах
бенчмарков.

**Дата:** 28 апреля 2026  
**Версия:** 1.0

---

## Общая оценка: **★★★★☆ (4.2 / 5)**

Решатель демонстрирует **отличные** результаты на большинстве практических задач. Основные сложности возникают на
классически плохо обусловленных полиномах (Wilkinson, экстремальный разброс коэффициентов).

## Подробный анализ

### Сильные стороны

| Область                        | Результат    | Комментарий                                  |
|--------------------------------|--------------|----------------------------------------------|
| **Случайные полиномы**         | Отлично      | Почти машинная точность                      |
| **Комплексные корни**          | Отлично      | Почти машинная точность                      |
| **Близкие / кластерные корни** | Очень хорошо | Хорошо справляется до 10-й степени           |
| **Малые коэффициенты**         | Отлично      | Крайне устойчиво                             |
| **Высокая кратность (чистая)** | Идеально     | `(x-r)^n` решается с ошибкой 0.00            |
| **Отсутствие NaN / Inf**       | Идеально     | Обработка критических точек работает надёжно |

### Слабые стороны

| Область                                   | Проблема                      | Серьёзность |
|-------------------------------------------|-------------------------------|-------------|
| **Полином Уилкинсона**                    | Ошибка растёт до 7e-6         | Средняя     |
| **Смешанная кратность (высокие степени)** | Ошибки до 2e-3                | Средняя     |
| **Экстремальный разброс коэффициентов**   | Большие абсолютные ошибки     | Высокая     |
| **Большие по модулю корни**               | Относительная ошибка до 1e-10 | Средняя     |

## Рекомендации

1. **Текущее состояние** — можно использовать для большинства реальных задач.
2. **Приоритетные улучшения**:
    - улучшить масштабирование переменной / балансировку коэффициентов для `extreme-coeffs` и `large-magnitude`;
    - добавить опциональный fallback на companion-матрицу + полировку для случаев типа Уилкинсона

## Заключение

**PolySolver** — это современная реализация метода Лагерра с неявной дефляцией.

**Отлично справляется с:**

- обычными полиномами;
- комплексными корнями;
- кратными корнями (чистыми);
- близкими корнями

**Требует доработки в:** крайне плохо обусловленных полиномах с огромным динамическим диапазоном коэффициентов.

**Итоговый вердикт:** готов к практическому использованию, с чёткими направлениями для дальнейшего развития.
